# Backend for Haigeia
## ExpressJS
We chose ExpressJS as our backend framework due to its simplicity, flexibility, and scalability. It also provides better performance over similar python backends such as FastAPI and Flask
To install dependencies please run
```
npm install
```
and to run the backend server please execute
```
node server.py
```

### Installations for modules such as Text to Speech currently need manual installation, please refer to respective directories for installation instructions
